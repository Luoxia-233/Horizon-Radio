# 灵感与想法

> 产品灵感参考，非 Lab 实验内容。未来正式产品开发时参考。

## 电台体验

- 不同时段（早晨/午间/傍晚/深夜）自动切换主持风格
- 歌曲间隙播放电台呼号（Station ID）
- 特定歌曲触发特定主持词（如经典曲目）
- 连续播放 N 首后主持人"休息"一段时间

## AI 主持人

- LLM 实时生成主持词
- TTS 合成语音
- 根据上一首歌风格和下一首歌风格生成过渡语
- 提及歌曲名称、艺术家信息的自然语言

## 天气与时间

- 接入天气 API，在间隔中播报天气
- 整点时间播报
- "深夜模式"使用更轻柔的主持语音

## 社交与分享

- 导出"电台录音"（含主持语音的播放记录）
- 自定义电台名称与呼号

## 音效增强

- 电台频率杂音效果（可选开关）
- 模拟 AM/FM 切换
- 信号不好时的失真效果
- "磁带"或"黑胶"质感音效

## 界面

- 《极限竞速：地平线》风格 UI
- 电台频率显示
- 霓虹/赛博朋克主题
- Mini Player 模式（悬浮小窗）
- 系统托盘模式

## 扩展可能性

- 远程控制（手机 Web 控制）
- Stream Deck 集成
- Discord Rich Presence
- 游戏内覆盖（Overlay）
