# 路线图

## Version 0.1 — MVP

> 目标：验证整体交互流程

- [ ] 浏览并加载本地音乐目录
- [ ] 显示播放列表
- [ ] 播放 / 暂停 / 下一首 / 上一首
- [ ] 支持随机播放
- [ ] 管理本地电台语音素材
- [ ] 每次切歌时随机播放一段主持语音
- [ ] 主持语音结束后继续播放下一首歌曲

**不做：** Fade、Ducking、事件系统、多电台

---

## Version 0.2 — 转场效果

> 目标：避免歌曲突然停止

- [ ] 音乐淡出（Fade Out）
- [ ] 主持语音播放（基于 v0.1）
- [ ] 下一首音乐淡入（Fade In）

---

## Version 0.3 — Audio Ducking

> 目标：模拟真实广播效果

- [ ] 主持人讲话时音乐不断开
- [ ] 音乐自动降低音量
- [ ] 主持结束后恢复原音量

---

## Version 0.4 — Radio Engine

> 目标：建立事件驱动的电台规则系统

- [ ] SongChanged 事件
- [ ] PlaylistStarted 事件
- [ ] PlaylistEnded 事件
- [ ] LongListening 事件
- [ ] ManualSkip 事件
- [ ] 全部广播行为由事件驱动

---

## Version 1.0 — 完整个人电台

> 目标：多频道、多主持人、完整电台体验

- [ ] 多个电台频道
- [ ] 不同主持人
- [ ] 不同语音素材池
- [ ] Station ID
- [ ] 节目片头
- [ ] 不同切歌策略

---

## 未来扩展（1.0+）

- [ ] AI 主持人
- [ ] 天气播报
- [ ] 时间播报
- [ ] 根据歌曲风格选择主持词
- [ ] 插件系统
- [ ] 网络广播
